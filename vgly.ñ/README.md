# Mundial 2026 en México - Proyecto de Implementa Software de Sistemas Informativos

## Descripción

Este sitio web tiene como objetivo proporcionar información relevante sobre la Copa Mundial de la FIFA 2026, que se celebrará en México, Estados Unidos y Canadá. El proyecto se enfoca principalmente en la participación de México, su historia en los mundiales, los jugadores más destacados, las sedes mexicanas y algunas curiosidades relacionadas con el torneo.

## Secciones del sitio

### Inicio
Se presenta una breve introducción sobre el Mundial FIFA 2026, incluyendo los países anfitriones y la importancia de México en esta edición.

### Historia
Se muestra información sobre la historia de México en las Copas del Mundo y algunos de sus momentos más importantes.

### Jugadores
Se presentan algunos jugadores mexicanos destacados y su contribución a la Selección Mexicana.

### Sedes
Se proporciona información sobre los estadios mexicanos que serán sede del Mundial 2026.

### Curiosidades
Se incluyen datos interesantes y curiosidades relacionadas con México y el Mundial FIFA 2026.

## Tecnologías utilizadas

- HTML
- CSS
- Bootstrap
- JavaScript
- JQuery

## Características

- Diseño responsivo mediante Bootstrap.
- Información organizada en cinco secciones principales.
- Uso de imágenes relacionadas con el Mundial 2026.
- Cuenta regresiva para el inicio del torneo.
- Funciones interactivas desarrolladas con JQuery.
